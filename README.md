## Introduction
Found-U is a web application designed for reporting and tracking lost property within educational institutions. It features a clean, mobile-first user interface and supporting real-time data synchronization.

## Pain Point
Traditional lost-and-found workflows in schools are often inefficient, disorganized, and difficult to track. 

## That's Why
Found-U addresses these bottlenecks by streamlining the reporting process for both losers and finders, enabling instant status updates via real-time Tracking Codes.

## Tech Stack (Version 0.1.2b)
- Next.js 16 (App Router) & React 19
- TypeScript 5.9
- Tailwind CSS 4
- Firebase 12 (Firestore, Storage, Authentication) & Firebase Admin SDK
- Bun Runtime
